# Kaggle Titanic: Machine Learning from Disaster

## Overview
This repository contains a machine learning solution for the famous [Titanic - Machine Learning from Disaster](https://www.kaggle.com/c/titanic) competition on Kaggle. The goal of this project is to build a predictive model that answers the question: "what sorts of people were more likely to survive?" using passenger data (i.e., name, age, gender, socio-economic class, etc.).

## Repository Structure
* **`Untitled.ipynb`**: The main Jupyter Notebook containing exploratory data analysis (EDA), data preprocessing, feature engineering, and the machine learning model building. *(Tip: You may want to rename this to something like `titanic_survival_prediction.ipynb` for better clarity).*
* **`train.csv`**: The training dataset containing passenger details and survival labels.
* **`test.csv`**: The testing dataset used to evaluate the model's performance.
* **`gender_submission.csv`**: A sample submission file provided by Kaggle demonstrating the correct prediction format.
* **`submission.csv`**: The final output file generated by the notebook, containing the survival predictions for the test dataset.

## Dataset Description
The data is provided by Kaggle and is split into two distinct groups:
* **Training set (`train.csv`)**: Used to build and train the machine learning models. It includes the target variable `Survived` (0 = No, 1 = Yes) along with various passenger features.
* **Test set (`test.csv`)**: Used to test the model's accuracy on unseen data. It does not include the ground-truth `Survived` column.

### Key Features
* **Pclass:** Ticket class (1 = 1st, 2 = 2nd, 3 = 3rd)
* **Sex:** Gender
* **Age:** Age in years
* **SibSp:** Number of siblings / spouses aboard the Titanic
* **Parch:** Number of parents / children aboard the Titanic
* **Ticket:** Ticket number
* **Fare:** Passenger fare
* **Cabin:** Cabin number
* **Embarked:** Port of Embarkation (C = Cherbourg, Q = Queenstown, S = Southampton)
